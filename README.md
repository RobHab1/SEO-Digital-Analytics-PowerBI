# SEO & Digital Analytics Dashboard

Business Intelligence solution developed in Microsoft Power BI to monitor SEO performance, website traffic and digital marketing KPIs through interactive dashboards, dimensional data modeling and Power Query ETL.

![Dashboard](images/dashboard.png)

---

# Project Overview

This project demonstrates how Business Intelligence techniques can transform digital marketing and SEO data into actionable business insights.

The solution combines data transformation, dimensional modeling and interactive reporting to provide decision-makers with a centralized view of website performance, search visibility and conversion metrics.

---

# Business Problem

Digital marketing teams collect data from multiple sources including websites, search engines and analytics platforms.

Without a centralized reporting solution it becomes difficult to:

- Monitor website performance
- Analyze search visibility
- Track traffic sources
- Measure conversions
- Identify performance trends
- Support data-driven marketing decisions

This dashboard addresses these challenges through an interactive Business Intelligence solution built in Microsoft Power BI.

# Data Preparation

The data was prepared using Power Query.

Main transformation steps included:

- Data import
- Data type conversion
- Filtering unnecessary records
- Handling missing values
- Removing duplicates
- Data standardization
- KPI calculations
- Data validation

![Power Query](images/power-query.png)

---

# Data Model

The solution follows a dimensional data model using a Star Schema.

The model separates fact and dimension tables to improve scalability, reporting performance and analytical flexibility.

Features include:

- Fact Table
- Customer Dimension
- Date Dimension
- Website Dimension
- SEO Dimension
- Traffic Source Dimension

![Star Schema](images/star-schema.png)
